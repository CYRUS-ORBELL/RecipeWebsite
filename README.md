# RecipeWebsite
HERE IS HOW TO SET THIS UP:
1. git clone
2. add .env file and add you information  add RecipeWebsite for database name
3. run in terminal: npm mysql ejs express env
4. Make a new database named Recipe Website in mysql 
5. in that database run SOURCE "the path to the recipeWebsite.sql file"
6. run node main.js
7  enjoy.
